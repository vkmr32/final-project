App Description - The following node server will ask a user to fill a form, which
is stored in a database, to fetch an image from the NASA APOD API for a particular day. 
Otherwise, the user can select to see the astronomy photo of the day.

API links - 
https://api.nasa.gov/planetary/apod
https://apod.nasa.gov/apod/astropix.html
